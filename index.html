Deep Learning Models on Acute Lymphoblastic Leukemia Blood Cancer Images

Conducted pioneering project on blood cancer detection using deep learning and medical imaging. Employed a dataset of ALL (Acute Lymphocytic Leukemia) blood cancer images which were divided into four sub categories of blood cancer and trained ten pre-trained deep learning models, including CNN, ResNet50, ResNet101, ResNet152, VGG16, VGG19, AlexNet, Xception, InceptionV3, and InceptionResNetV2 . Utilized color thresholding-based segmentation and data augmentation for robust model training. Achieved exceptional results, with CNN, ResNet50, and ResNet101 outperforming others in accurately classifying blood cancer stages. This project demonstrates the potential for implementing efficient machine-based tools in medical diagnosis, paving the way for revolutionizing blood cancer detection.
